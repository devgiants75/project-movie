### 메인 페이지
- *네비게이션*
  - 홈 아이콘 (마우스 클릭 시 메인 페이지로 이동)
  - 상영 중, 상영 예정, 고전 명작 링크
  - 검색창 (감독/배우 입력 시 상세페이지로딩)
  - 로그인 버튼 (마우스 클릭 시 로그인 페이지로 이동)
  - 회원가입 버튼 (마우스 클릭 시 회원가입 페이지로 이동)
- *영화 예고편*
  - 자동 재생되는 상영 중인 영화 예고편(음소거로 자동재생 5~10초 후 다음 예고편 자동넘김)
    ㄴ 좌우 버튼을 통한 다음/이전 예고편 재생
    ㄴ 예고편 박스 중앙 하단 부분에 스와이프 기능
    ㄴ
- *영화 목록*
  - 좌우 화살표 버튼을 통한 다음/이전 영화
  - 이미지 링크 4개 (각 이미지 클릭 시 해당 영화 소개 페이지로 이동)
### 로그인 페이지
- *로그인 양식*
  - 이메일 입력
  - 비밀번호 입력
  - 로그인 버튼
  - 회원 가입 링크 (마우스 클릭 시 회원 가입 페이지로 이동)
  - 아이디 저장, 자동로그인 체크박스 기능
  - 영화 포스터 랜덤 표시
### 회원 가입 페이지
- *회원 가입 양식*
  - 사용자 닉네임 입력
  - 이메일 주소 입력
  - 비밀번호 입력 (특수 문자 포함, 14자 제한)
  - 비밀번호 확인 입력
  - 회원 가입 버튼
  - 이미 회원이신 경우 로그인 링크 (마우스 클릭 시 로그인 페이지로 이동)
### 영화 페이지
- *헤더*
  - 네비게이션 상영중 상영예정 고전명작 밑 상영중인영화 문구가 있으며 검색창 밑에 상영중,상영예정,고전명작 링크가 있음(문구 클릭시 해당링크에 맞는 문구로 변경)
- *영화 목록*
  - 4x2 영화 포스터 배치되어있으며 각 포스터 밑에는 영화제목, 영화평점 확인 가능하며 리뷰보기 링크버튼이 존재함
    ㄴ 포스터 이미지 클릭시 영화소개 페이지로 이동 리뷰보기 클릭시 평점/리뷰 페이지로 이동
### 영화 소개 페이지
- *영화 소개*
  - 네비게이션 밑 영화 트레일러 이미지컷 있으며 투명도 조절한다음 이미지 위에 오버된 텍스트 텍스트 내용에는 영화이름,원어,개봉년월/국가,장르,상영시간,등급이 포함
  - 평점 표시,출연/제작정보, 리뷰보기&찜하기 링크가 있으며 해당 영화의 대략적인 줄거리 확인 가능
  - 좌우 화살표 버튼을 통한 트레일러 or 스틸컷 확인가능
### 평점 & 리뷰 페이지
- *영화 소개*
  - 45번줄 기능과 동일
  - 인스타 피드 비슷한 디자인 카드 섹션5개 존재하며 좌우 화살표 버튼을 통한 다음 이전 리뷰 확인 가능
### 통합 검색 결과창
- *필모그래피 (출연/연출)*
  - 메인홈페이지 검색창에 입력한 감독 | 배우 연출 | 출연한 영화 표시
  - 인물 사진이 있으며 출연 | 연출 한 영화 포스터 이미지 링크,리뷰보기링크가 존재함
    ㄴ 포스터이미지 링크 클릭시 해당 영화 소개 페이지로 이동 / 리뷰보기 클릭시 해당 영화 평점/ 리뷰 페이지로 이동
### 회원 정보 수정 페이지
- *회원 정보 수정 양식*
  - 닉네임 변경
  - 프로필 이미지 변경
  - 이메일 주소 표시
  - 작성한 리뷰 ,찜한 영화 좋아요 링크
회원
- 접근 가능한 페이지: 모든 페이지
- 사용 가능한 기능: 모든 기능
비회원
- 접근 가능한 페이지: 메인, 로그인, 회원가입, 영화, 영화 소개, 평점/리뷰, 상세 검색
- 각 페이지에서 사용 가능한 기능:
  - 메인: 구현된 모든 기능 사용 가능
  - 로그인: 구현된 모든 기능 사용 가능
  - 회원가입: 구현된 모든 기능 사용 가능
  - 영화: 구현된 모든 기능 사용 가능
  - 영화 소개: 구현된 모든 기능 사용 가능
  - 평점/리뷰: 리뷰 공유 버튼, 이전&다음 리뷰 버튼
  - 상세 검색: 구현된 모든 기능 사용 가능
- 접근 불가능한 페이지: 개인 정보 수정, 청소년관람불가등급인 영화 소개,평점/리뷰페이지는 로그인 필요
- 접근 가능하지만 사용할 수 없는 기능:
  - 평점/리뷰 페이지: 좋아요 기능, 찜하기, 리뷰 등록
필요한 데이터 정보
1. *영화 정보:*
   - 제목(한글,영어)
   - 장르
   - 감독
   - 배우 목록
   - 제작사 or 배급사
   - 개봉일
   - 국가
   - 포스터 이미지
   - 예고편 동영상 URL
   - 영화 줄거리
2. *리뷰 정보:*
   - 작성자 닉네임
   - 리뷰 내용
   - 별점
   - 작성일
   - 좋아요 수
   - 공유
3. *사용자 정보:*
   - 사용자 아이디
   - 비밀번호
   - 이메일
   - 프로필 사진
   - 작성한리뷰
   - 찜한영화
   - 좋아요 표시한 영화
4. *검색 및 필터링을 위한 데이터:*
   - 영화 및 리뷰에 대한 키워드
   - 영화별 장르 정보
   - 사용자가 작성한 리뷰에 대한 키워드 등
5. *세션 및 인증을 위한 데이터:*
   - 사용자 세션 정보 (로그인 상태 등)
   - 사용자 권한 정보 (일반 사용자, 관리자 등)
